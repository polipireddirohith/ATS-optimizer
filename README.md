

## ✨ Features

- **🎯 Intelligent Scoring**: Advanced ATS compatibility scoring (0-100) based on keywords, skills, experience, and formatting.
- **🔍 Gap Analysis**: Detailed breakdown of missing critical requirements and optimization opportunities.
- **👔 Recruiter Insights**: HR-specific suitability verdicts with risk assessment and experience check.
- **💡 Smart Improvements**: Actionable suggestions for keyword placement and bullet point rewrites.
- **📄 Document Support**: Handles PDF and DOCX formats seamlessly.
- **🎨 Premium UI**: Modern SaaS aesthetic with multi-theme support (Midnight, Emerald, Sunset) and interactive background effects.
- **⚡ Real-time Optimization**: Live extraction and analysis of job descriptions.

## 🛠️ Tech Stack

- **Backend**: Python, Flask
- **Parsing**: pdfplumber, python-docx, PyPDF2
- **Frontend**: Vanilla Javascript, CSS3 (Custom Variables, Grid, Animations)
- **Design**: Premium Minimal Enterprise UI / Glassmorphism

## 🚀 Getting Started

### Prerequisites

- Python 3.8 or higher
- Pip (Python Package Manager)

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/ats-optimizer.git
   cd ats-optimizer
   ```

2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the application**:
   ```bash
   python app.py
   ```

4. **Access the web app**:
   Open `http://localhost:5000` in your browser.
